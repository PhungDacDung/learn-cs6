Learn Javascript ES6

ES6 là chữ viết tắt của ECMAScript 6. ES6 ra đời vào năm 2015 nên cái tên ES2015 được lấy làm tên chính thức với nhiều tính năng mới học hỏi các ngôn ngữ cấp cao khác, hy vọng dần theo thời gian Javascript trở thành một ngôn ngữ lập trình hướng đối tượng.

Các chức năng mới của ES6

  • Let & Const
  • Arrow function
  • Rest parameters
  • Destructuring Assignments
  • Default Parameters
  • Template String
  • Classes
  • Modules
  • Spead

Nội dung

  • Let & Const
    • Let: thường dùng để khai báo các biến mang tính chất tạm thời, nghĩa là nó chỉ sống trong một phạm vi hoạt động của khối đó thôi, không sử dụng qua vị trí khác. block scope được đánh dấu trong phạm vi cặp ngoặc nhọn {}. Và khi thoát khỏi block scope thì nó cũng bị mất đi.
    • var a = 12;
      var b = 20;
      if (a < b)
      {
      var tmp = a;
      a = b;
      b = tmp;
      }
      console.log("a: " + a);
      console.log("b: " + b);
      console.log("tmp: " + tmp); // đoạn này lỗi
    • Const: cũng giống như let, const cũng là block scope const không cho phép người dùng khai báo nhiều lần trong cùng scope, và cũng không cho phép người dùng cập nhật hay thay đổi giá trị của biến.
    • const greeting = "say Hi";
      greeting = "say Hello instead"; // error : Assignment to constant variable.
      ------------------------------------------------
      const greeting = "say Hi";
      const greeting = "say Hello instead"; // error : Identifier 'greeting' has already been declared
  • Arrow function
    • Arrow function - còn được gọi là "fat arrow", là cú pháp được mượn từ CoffeeScript (một ngôn ngữ chuyển tiếp), cú pháp này là cách ngắn gọn hơn dùng để viết function. Ở đây sử dụng kí tự =>, trông giống như một mũi tên.
    • +Trong trường hợp nhiều tham số
      // ES5
      var multiply = function(x, y) {
      return x * y;
      };
      // ES6
      var multiply = (x, y) => { return x * y };
    • + Trong trường hợp 1 tham số
      //ES5
      var phraseSplitterEs5 = function phraseSplitter(phrase) {
      return phrase.split(' ');
      };
      //ES6
      var phraseSplitterEs6 = phrase => phrase.split(" ");
  • Destructuring Assignments
    • Destructuring assignment là một biểu thức JavaScript dùng để lấy ra (destructure) giá trị của một hay nhiều phần tử rong mảng hoặc đối tượng đồng thời gán các giá trị này cho các biến cho trước.. Điều này có thể làm giảm đáng kể các dòng mã cần thiết để thao tác dữ liệu trong các cấu trúc này. Có hai loại Destructuring: Destructuring Objects và Destructuring Arrays.
    • Destructuring Objects
       let user = {
      name: 'Name',
      age: 20,
      address: 'HN'
      }
      Theo cách truyền thống thì chúng ta sẽ lấy ra những giá trị như cú pháp sau:
      let name = user.name // name = 'Name'
      let age = user.age // age = 20
      let job = user.address // address = 'HN'
      Nhưng với việc sử dụng object destructuring chúng ta có thể get được những giá trị ấy miễn là trùng tên của thuộc tính trong object là được:
      let { name, age,address } = user // name = 'Name', age = 20, address = 'HN'
  • Rest parameters
    • Rest Parameters dịch theo tiếng Anh chuyên ngành công nghệ thông tin có nghĩa là tham số còn lại, điều này có nghĩa là bạn có thể khai báo một hàm với số lượng tham số không xác định, đây là một tính năng mới khiến Javascript ngày càng trở nên mạnh mẽ hơn.
    • let functionName = (param1, param2, ...other) => { }
  • Default Parameters
    • Default Parameter là giá trị mặc định của tham số khi truyền vào các function. Sử dụng khi định nghĩa hàm có những tham số không bắt buộc phải nhập
    • function logger(log, isAlert = false){
      if (isAlert) return alert(log);
      console.log(log);
      }
      logger("Message...",true);
      // hiển thị thông báo alert
  • Template String
    • Template Strings là một cú pháp mới để khai báo String trong Javascript được giới thiệu trong ES2015/ES6 Nó cho phép chúng ta sử dụng multi-line string, sử dụng biến, biểu thức, hàm bên trong string mà không phải thông qua phép cộng string.
    • const courseName ="Javascript"
      const description = `Course name: ${courseName}`;
      console.log(description);
      // Course name: Javascript
      • ES6 template string sử dụng dấu backtick để biểu diễn.
      ES6 template string giúp ghép nối string đơn giản hơn, không cần phải sử dụng toán tử + hay phương thức concat nữa.
      ES6 template string giúp sử dụng đồng thời dấu nháy đơn và nháy kép mà không cần dùng kí tự \.
      ES6 template string giúp viết string trên nhiều dòng đơn giản hơn.
      ES6 template cho phép viết biểu thức hoặc gọi hàm ngay trong string.
  • Classes
    • Trong ES6 đã hỗ trợ chúng ta khai báo một đối tượng theo chuẩn OOP, bằng cách sử dụng từ khóa class.
    • class Employee {
      setName (name) {
      this.name = name;
      }
      getName () {
      return this.name;
      }
      setAge (age) {
      this.age = age;
      }
      getAge () {
      return this.age;
      }
      };
  • Modules
    • Sử dụng export riêng lẻ ở cuối file module. Cách sử dụng này được gọi là "named export" do ta định nghĩa tên của object đang được export trong khi đang export.
      function foo() { return 'foo'; }
      function bar() { return 'bar'; }
      export { foo, bar };
      Ta có thể sử dụng object được export từ các file khác như sau:
      import {foo, bar} from 'foobar';
      foo();
      bar();
      import * as lib from 'foobar';
      lib.foo();
      lib.bar();
    • Một format khác khi sử dụng "named export" là ta có thể export objects/function khi ta tạo ra chúng. Cấu trúc sau có phần tiện hơn so với ở ví dụ 1.
       export function foo() { return 'foo'; }
      export function bar() { return 'bar'; }
  • Spead
    • Cú pháp spread đơn giản được biểu diễn bởi dấu 3 chấm: ...
      Spread Syntax cho phép duyệt qua các phần tử và truyền vào phương thức như các đối số
    • var mid = [3, 4];
      var arr = [1, 2, ...mid, 5, 6];
      console.log(arr);
      // kết quả: [1,2,3,4,5,6]
